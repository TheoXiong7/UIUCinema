About

CS410 Final Project - Fall 2025

Overview

This movie search engine combines traditional information retrieval with modern semantic search techniques to provide accurate and relevant movie recommendations based on natural language queries.

The system is built on the IMDB Movie Dataset and implements a hybrid search approach that balances keyword matching with semantic understanding.

Technical Approach

1. BM25 Ranking

Traditional lexical search using BM25Okapi algorithm with tuned parameters (k1=1.2, b=0.75) for optimal term frequency and document length normalization.

2. Semantic Embeddings

Neural sentence embeddings using SentenceTransformers (all-MiniLM-L6-v2) to capture semantic meaning beyond exact keyword matches.

3. Hybrid Fusion

Score normalization and weighted combination (alpha=0.65) of both approaches to leverage the strengths of lexical and semantic search.

4. Text Preprocessing

Advanced cleaning, lemmatization using spaCy, and stopword removal to improve retrieval quality and reduce noise in the corpus.

Dataset

The system uses the IMDB Movie Dataset enriched with keyword extraction using:

  • KeyBERT for contextualized keyword extraction
  • YAKE for unsupervised keyword extraction
  • Sentence Transformers for semantic keywords

Evaluation

Precision@10

Measures the proportion of relevant movies in the top 10 results

nDCG@10

Normalized Discounted Cumulative Gain for ranking quality assessment
